**BMI Calculator (Python Project)**
Overview
The BMI Calculator is a Python application that computes Body Mass Index using user-provided height and weight, and classifies it into standard health categories. It includes both a command-line version and a GUI version built with Tkinter.

**Key Features**
* BMI calculation and category classification
* Input validation and error handling
* GUI implementation using Tkinter
* Data storage using CSV
* Basic data visualization using Matplotlib

**Technologies Used**
* Python
* Tkinter
* CSV
* Matplotlib
